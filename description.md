### Use Case: 1. 회원가입
| Actor Action | System Response|
|---|---|
| 1. 회원 가입 버튼 클릭||
|| 2. 회원 가입 페이지로 이동 (ID, 비밀번호, 전화번호, 결제 수단, 선호 자전거 유형(일반/전기))|
| 3. 정보 입력 후 유효성 검사 버튼 클릭 ||
|| 4. 정보 유효성 검사 후 가입 버튼 활성화|
| 5. 가입 버튼 클릭||
|| 6. 가입 완료 안내 화면 표시 (로그인 화면으로 이동)||

### Use Case: 2. 로그인
| Actor Action | System Response|
|---|---|
|1. 로그인 버튼 클릭||
||2. 로그인 페이지 이동(ID, PW)|
|3. 정보 입력 후 로그인 버튼 클릭||
||4. 정보 확인 후 로그인|

### Use Case: 3. 회원 탈퇴 요청
| Actor Action| System Response|
|---|---|
| 1. 회원이 “회원 탈퇴” 버튼 클릭 ||
|| 2. 탈퇴 확인 페이지로 이동|
| 3. 탈퇴 확인서 동의||
|| 4. 모든 이용 권한 및 데이터 삭제 후 로그인 화면으로 이동|
